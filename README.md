# NBA Position Aging Analysis

## Research Question
Which NBA positions age better, and are these differences related to how key skills (athleticism, shooting, playmaking) decline over time?

## Data Sources
- nba_api: Season stats for 2010-2024
- Kaggle: Position and advanced stats (PER, BPM, VORP, WS)

## Dataset
- 3570 player-season observations
- 5 positions: PG, SG, SF, PF, C
- Filter: 1000+ minutes per season

## AI Tool Usage
Claude was used for data collection code, debugging, and EDA guidance.
